# Actions_with_files

Duplicate_file_deleter - Простенькая удалялка дубликатов файлов на go, удаляет файлы с (1,2,...n) в папке, оставляя тем самым только оригинал файла.
Music_File_Sorter - сортировщик музыкальных файлов по папкам по типу : AC-DC - Thunderstruck.mp3 => папка AC-DC в которую будет добавлен этот файл и все последующие файлы этой группы будут добавлены в эту папку.
File_Unpacker - распаковка всех папок в корневую папку.
